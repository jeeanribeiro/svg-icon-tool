# Security Policy

## Supported versions

| Version | Supported |
| :------ | :-------- |
| 3.x     | yes       |
| < 3.0   | no        |

## Reporting a vulnerability

Please do not open a public issue for security problems. Instead, use
[GitHub private vulnerability reporting](https://github.com/jeeanribeiro/svg-icon-tool/security/advisories/new)
or email contact@jeanribeiro.dev.

You can expect an acknowledgment within a few days. Please include a
proof-of-concept SVG input where applicable — the tool parses untrusted
SVG, so parser-level issues (XML entity tricks, pathological inputs causing
unbounded work) are in scope.
